# 🧑‍🚀 Hello, Astronaut!

Welcome to the official Astro blog starter template. This template serves as a lightweight, minimally-styled starting point for personal websites, blogs, and portfolios built with Astro.

This site uses Markdown files for content which are rendered via the Astro Content API. Edit `src/content/pages/index.en.md` to update this page.

## Getting started

- Edit this page in `src/content/pages/index.en.md`
- Edit the header items in `src/components/Header.astro`
- Add your name to the footer in `src/components/Footer.astro`
- Check out the included blog posts in `src/content/blog/`
- Customize the blog post layout in `src/layouts/BlogPost.astro`
